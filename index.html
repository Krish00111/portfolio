Objective: Provide public safe-zone data using geolocation and NDRF locations.
Methodology: Frontend (HTML/CSS/JS) + maps, geolocation API, backend for location data.
Outcome: Interactive map and quick guidance for users during coastal threats.
Challenges & Solutions: Integrating live location accuracy — solved by smoothing updates and throttling API calls.

Challenges: Handling duplicate articles — solved with hashing + timestamp rules.
